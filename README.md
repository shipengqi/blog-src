# blog-src

My blog source code.

## requirements
- NodeJs 
- Hexo(全局)

## Usage
```bash
#安装依赖
yarn install # or npm install

#启动
./run.sh
```

**Options**
```bash
Options:
  --help
    Display this help screen

  --start
    启动服务

  --port
    指定监听的端口

  --deploy
    部署到github
```
